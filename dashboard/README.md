# cokit-ultra - Dashboard

Simple status dashboard for cokit-ultra services.

## Build

```bash
docker build -t cokit-ultra-dashboard .
```

## Run

```bash
docker run -p 8080:80 cokit-ultra-dashboard
```

## Features

- Real-time plugin status
- Token savings tracker
- FastContext queue status
- Service health checks
